Lab 16 - JSON and APIs

Challenge

Our challenge is to use the XKCD API to retrieve and display a random comic using jQuery and JSON.

Problems

The biggest challenge was dealing with CORS errors when trying to fetch data from the XKCD API using AJAX. The browser blocked the request because the API didn't allow cross-origin access. I resolved this by using a public CORS proxy to successfully load the data.

Reflection

This lab helped me better understand how APIs work in the browser and how to handle issues like CORS. I practiced using jQuery's AJAX functions to load external data, parse JSON, and dynamically update the page with comic images and info from XKCD.
